# Screenshots

This directory is reserved for product screenshots used by the README and project documentation.

Recommended screenshots to add:

- `login.png` - public login page.
- `dashboard.png` - asset dashboard with sample, non-sensitive data.
- `stocks.png` - stock holdings and transaction history with sample, non-sensitive data.
- `metrics.png` - metrics visualization page with sample, non-sensitive data.
- `mobile-dashboard.png` - mobile view of the asset dashboard.

## Screenshot Guidelines

- Use demo or sample data only.
- Do not include real financial records, email addresses, access tokens, cookies, or Supabase project identifiers.
- Capture desktop screenshots at 1440px wide when possible.
- Capture mobile screenshots around 390px wide when possible.
- Keep images compressed enough for GitHub to load quickly.

After screenshots are added, update the Screenshots section in the root README to embed them.
